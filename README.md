### Kids Smart Card

Kids Smart Cards with sound accompaniment

App based on:

- MVVM (Jetpack ViewModel + Kotlin SharedFlow)
- Repository based on Coroutines Flow and suspend functions
- Koin DI
- Room and OkHttp in data module
- Splash screen with logo
